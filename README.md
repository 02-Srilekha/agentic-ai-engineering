# AI Agent Engineering Notes

This repository contains structured notes, experiments, and learnings from a 5-day training program focused on AI Agents, context engineering, and modern software development lifecycle (SDLC) in the agentic era.

---

## 🎯 Objective

To understand how software engineering is evolving from:

> Building software → Building systems that generate software

---

## 📚 What This Repo Covers

- AI Agents fundamentals
- Context Engineering
- Agentic Software Development Lifecycle (SDLC)
- Multi-agent systems
- Evaluation & observability
- Prompting vs system design
- Real-world agent workflows

---

## 🧱 Course Structure

The notes are organized by day:

- Day 1: Introduction to agents and SDLC changes
- Day 2: Tool use and interoperability
- Day 3: Memory, context, and agent capabilities
- Day 4: Evaluation, safety, and guardrails
- Day 5: Production deployment and system design

## 🔑 Key Idea

Modern AI engineering is not about writing better prompts.

It is about designing **reliable systems that consistently produce correct outputs using AI agents.**

---

## 🧠 Core Themes

- Context Engineering > Prompt Engineering
- Harness design > Model tuning
- Evaluation > Generation
- Systems thinking > Single prompts
- Reliability under autonomy

---
## ⚙️ Example Agent Workflow

Input (User Intent)
→ Planning Agent
→ Tool Execution (APIs / Functions)
→ Validation Layer (Guardrails)
→ Final Output
